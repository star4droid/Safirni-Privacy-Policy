سياسة الخصوصية - سفرني

في تطبيق "سفرني" المخصص لطلب تذاكر السفر داخل السودان، نلتزم بحماية خصوصيتك وبياناتك. تهدف هذه السياسة لشرح البيانات التي نجمعها وكيفية استخدامها وحمايتها.

البيانات التي نجمعها

  • الاسم ورقم الهاتف: تُستخدم لتأكيد الطلب والتواصل معك عند الضرورة.
  • بيانات الرحلة: وتشمل خط السفر (من – إلى) واليوم المطلوب فقط، دون اختيار مقعد محدد.
  • بيانات الجهاز الأساسية: مثل نوع الجهاز وإصدار النظام لتحسين الأداء وعلاج الأخطاء.

الأذونات التي نطلبها ولماذا

  • INTERNET: لعرض خطوط الرحلات، إرسال طلبات الحجز، واستقبال تأكيد الطلب.
  • ACCESS_NETWORK_STATE: للتحقق من وجود اتصال بالإنترنت قبل إرسال الطلب.

كيف نستخدم البيانات

  • معالجة طلبات السفر وفق البيانات التي يرسلها المستخدم.
  • التواصل مع المستخدم لتأكيد الطلب أو إبلاغه بأي تحديث.
  • تحسين أداء التطبيق ومعالجة المشاكل.

الأمان

نستخدم معايير حماية مناسبة لضمان سرية بيانات المستخدم. كل الاتصالات مع الخادم مشفّرة، ولا نقوم بتخزين أي بيانات حساسة غير ضرورية.

عدم مشاركة البيانات

لا نشارك بيانات المستخدم مع أي جهة خارجية إلا شركات النقل المعنية بمعالجة الطلب.

Privacy Policy - Safarni

"Safarni" is a Sudanese travel-ticket request application. We are committed to protecting your privacy. This policy explains what data we collect and how it is used.

Information We Collect

  • Name and Phone Number: Used to confirm the request and contact you when needed.
  • Trip Information: Route (from–to) and the selected day only. No seat selection is required.
  • Basic Device Information: Such as OS version and device type for improving performance and fixing issues.

Permissions We Request

  • INTERNET: Required for sending travel requests and receiving confirmations.
  • ACCESS_NETWORK_STATE: Used to verify internet connectivity before sending data.

How We Use the Data

  • To process travel requests based on the user’s submitted details.
  • To contact the user for confirmation or important updates.
  • To improve app performance and diagnose issues.

Data Security

We implement technical measures to protect user data. All communication with the server is encrypted, and no unnecessary sensitive data is stored.

No Data Sharing

We do not share user data with third parties except transport companies involved in fulfilling the travel request.
